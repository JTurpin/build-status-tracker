  • "/"
    • GET

  • "/update"
    • POST
    • Fields
      • Name *Required*
      • BuildVersion *Required*
      • BuildPromoted bool
        • format expected
          • true
          • false
      • Description string
      • LastBuildStatus *Required*
      • CallbackURL string
    • Example:
      • curl -vvv -XPOST -H'Content-Type: application/json' -d '{"Name":"autobot","Description":"Its True","BuildVersion":"654","LastBuildStatus":"Fail","BuildPromoted":true,"CallbackURL":"http://google.com"}' http://localhost:7080/update
  • "/delete"
    • POST
    • Example:
      • curl -v -XPOST -H'Content-Type: application/json' -d '{"Name":"MattSucks","Description":"Its True","BuildVersion":"420","LastBuildStatus":"Fail","BuildPromoted":true,"CallbackURL":"http://google.com"}' http://localhost:7080/delete
